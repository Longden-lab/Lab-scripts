How to Use
🚀 Uploading a Script
  1. Navigate to the correct folder (e.g., image_processing/ or Bioinformatics/)
  2. Click "Add file" → "Upload files"
  3. Drag and drop your script file
  4. Add a short description at the bottom under "Commit changes"
  5. Click "Commit changes" — done!
📁 Creating a New Folder
  1. Navigate to where you want the new folder
  2. Click "Add file" → "Create new file"
  3. In the filename box, type your folder name followed by / and a filename:
    e.g. new_folder_name/README.md
    Typing / automatically tells GitHub to treat it as a folder
  4. Add a short description of what the folder is for in the file body
  5. Click "Commit changes"
💡 The README.md you create inside will automatically display as the folder's description — always fill it in with a sentence about what belongs there.
File Naming Convention

Please name your files clearly using this format:

topic_yourname.ext

Example:  PAdiameter_Gyu.R
