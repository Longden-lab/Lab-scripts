How to Use
🚀 Uploading a Script
  1. Click the correct category card above (Image Processing or Bioinformatics)
  2. Navigate into the specific subfolder for your script type
  3. Click "⬆ Upload Script" on that page, or go directly to GitHub and click "Add file" → "Upload files"
  4. Drag and drop your script
  5. Add a short description under "Commit changes" and click "Commit changes"
📁 Creating a New Folder
  1. Navigate to the correct category page using the cards above
  2. Scroll down to the "Need a new subfolder?" guide on that page
  3. Follow the steps there — or go directly to GitHub, click "Add file" → "Create new file"
  4. In the name box type: your_folder_name/README.md (the / creates the folder)
  5. Click "Commit changes"
⚠️ Always use _ instead of spaces in folder and file names — e.g., new_folder not new folder
File Naming Convention

Please name your files using this format:

topic_yourname.ext

Example:  PAdiameter_Gyu.R  |  rep_image_export_Jane.ijm  |  batch_preprocess_Tom.m

💡 Not sure which folder to use? Click a category card above — each subfolder page has a description of what belongs there.
