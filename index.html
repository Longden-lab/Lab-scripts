🚀 How to Upload a Script

  1. Go to the GitHub repository and log in
  2. Navigate into the correct folder (e.g., bioinformatics/R/)
  3. Click "Add file" → "Upload files"
  4. Drag and drop your script file
  5. Add a short description at the bottom under "Commit changes"
  6. Click "Commit changes" — done!

📋 File Naming Convention

Please name files as: topic_method_yourname.ext
Example: cellsegmentation_watershed_jane.py
