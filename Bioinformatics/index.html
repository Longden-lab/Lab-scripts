📁 How to create a new folder inside Bioinformatics
  1. Go to this folder on GitHub
  2. Click "Add file" → "Create new file"
  3. In the name box type: your_folder_name/README.md
    The / automatically creates the folder
  4. Write a short description in the file body
  5. Click "Commit changes"
⚠️ Use _ not spaces — e.g., RiboTag_seq not RiboTag seq
